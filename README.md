# simple-server
Simple HTTP Server in python.

## Features
- Directory listing with modern UI.
- File uploads via the browser.
- Create folders and delete files from the UI.
- Download directories as a zip archive.
- List running server instances via `list`.
- Optional password protection with 30-minute login sessions.

## Usage
```bash
python src/simpleserver.py [address:port] [path] [--password <value> | -pwd <value>]
```
